# Adaptive Behavioral Intelligence System for Resource Utilization Optimization

## Project Overview

This project presents an adaptive GPU resource monitoring and optimization system designed for AI workloads. The system monitors GPU behavior in real-time, detects stress conditions such as overload and thermal spikes, and applies workload-level optimization techniques.

The system follows a closed-loop architecture:
Monitoring → Analysis → Decision → Optimization

## Tech Stack

Frontend:
- React (Vite)
- React Router
- CSS

Backend (Upcoming):
- Node.js
- Express.js

## Features Implemented

- Authentication System (Login)
- Permission-Based Access Control
- Protected Routing
- GPU Metrics Dashboard UI
